# Pure Peptides

Research-grade peptide supplier website. Built with clean HTML + Tailwind CSS + vanilla JS.

## Local Development

```bash
python3 -m http.server 8000
# Open http://localhost:8000
```

## Deployment

This site deploys to GitHub Pages from the `main` branch. Push to `main` to trigger deployment.

**Live:** https://kignleon.github.io/direct-peptides/

## Tech Stack

- Semantic HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Inter font (Google Fonts)
- Font Awesome 6 (CDN)

## Structure

```
/
├── index.html          Main page
├── css/style.css       Brand system + components
├── js/script.js        Interactions (menu, modal, FAQ, scroll-reveal)
├── assets/             Logo + images
├── skills/             Build skills reference
├── .nojekyll           Disable Jekyll processing
├── robots.txt          Search engine directives
└── README.md           This file
```
